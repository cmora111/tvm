TermForge

A Tkinter virtual macropad for Linux/X11.

Features
  • Select a target window with libxdo
  • Launch terminal commands
  • Send text to a selected terminal window
  • Keep commands in a simple Python config file

Install

python3 -m pip install .

Run

termforge

Config

User config lives at ~/.config/termforge/config.py.
